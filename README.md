# PeekN

PeekN is a category-based product review and discovery platform where users can explore video reviews, product insights, and community-driven recommendations.

## Features

- Product/category-based review system
- Video and image upload support
- Cloudinary media integration
- Responsive UI for mobile and desktop
- Dynamic content interaction
- Deployed live using Render

## Tech Stack

- Python
- HTML/CSS
- Tailwind CSS
- Cloudinary
- Render
- Git & GitHub
